1. Название проекта: Telegram бот для отображения курса доллара к аргентинскому песо. 
2. Описание: Разработать Telegram бота, который будет предоставлять пользователю актуальный курс доллара к аргентинскому песо. Курс должен быть в реальном времени и подтягиваться с внешнего сайта. 
3. Функционал: 
   - Бот должен иметь команду /start для запуска и приветственного сообщения. 
   - Бот должен иметь команду /usdars для получения текущего курса доллара к аргентинскому песо. 
   - Бот должен подтягивать актуальный курс с внешнего сайта. 
4. Технические требования: 
   - Использование языка программирования Python. 
   - Использование библиотеки python-telegram-bot для работы с Telegram API. 
   - Использование библиотеки requests для получения данных с внешнего сайта. 
   - Деплой бота на платформе, поддерживающей Python (например, Heroku). 
5. Дополнительные требования: 
   - Реализация обработки ошибок при получении данных с внешнего сайта. 
   - Реализация кэширования данных для уменьшения запросов к внешнему сайту. 
6. Временные рамки: Разработка и тестирование бота должны быть завершены в течение 1 недели с момента начала работы над проектом
